WeirdChromie
===

ChromieCraft (3.3.5) addon. Quality-of-life patches for the things the server gets in your way about.

Features
---
**Chat & Spam**
* Silence ChromieCraft server spam (queue announcers, leaderboards, tips, server notices, joyous journeys, etc.).
* Silence specific Dalaran ambient NPCs chatter

**Automation**
* Auto-skip routine NPC gossip.
* Auto-confirm the bind-on-pickup loot popup when solo. Skipped in a party/raid and for items the pass rules would pass on.
* Auto-delete Minigob Manabonk spam mail and destroy the attached Mischief Maker. Skipped when bags are full.
* Auto-dismount flying mounts when an attack is blocked by "Can't attack while mounted."
* Disable right-click to attack alive enemies, so right-dragging the camera over a mob no longer pulls or swaps target. Default off.

**Loot Rolls**
* Auto-roll (pass/greed/need) on the BoP jewelcrafting designs from Northrend dungeon bosses.
* Auto-roll (pass/greed/need/disenchant) on uncommon BoE drops, with optional Greed-instead-of-Disenchant for weapons and shields.
* Auto-roll (pass/greed/need) on Frozen Orb from heroic 5-man bosses. Only fires inside a 5-man dungeon.
* Auto-pass group loot rolls on Heavy Frostweave Bandage and the four Northrend mob-drop cooking recipes.

**Encounter Helpers**
* Drake essence button (Oculus): movable. Click uses the held essence; if mounted on a drake, click leaves the vehicle instead.
* Self-cast Revivify and Life Burst on the Wyrmrest Skytalon (Eye of Eternity P3).

**Other**
* Fix RDF cooldown frame showing stale party-member status when the dungeon finder opens before deserter/cooldown auras have synced. Currently not working.

Slash commands
---
* `/wc` opens the options panel.
* Hold Ctrl when opening a gossip frame to bypass auto-skip for one interaction.

Options
---
Esc -> Interface -> AddOns -> WeirdChromie
